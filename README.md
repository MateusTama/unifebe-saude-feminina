# Minha Saúde Feminina

Este é um projeto de **Curricularização da Extensão** desenvolvido por estudantes da **UNIFEBE**. O objetivo é criar uma aplicação mobile voltada à saúde feminina, focando na disseminação de informações confiáveis baseadas em diretrizes do sistema público de saúde brasileiro.

## O Projeto

O aplicativo permite que usuárias acompanhem o seu ciclo menstrual, registrem sintomas diários e acessem artigos educativos organizados por temas como Menstruação, Gravidez e Contraceptivos.

## Estrutura do Monorepo

```
SistemaSaudeFeminina/
├── frontend/   # Aplicação Mobile (React Native + Expo)
├── backend/    # API REST (Python + Flask, padrão MVC)
└── sql/        # Scripts de modelagem e criação do banco de dados
```

| Camada | Tecnologias |

| **Frontend** | React Native, Expo, Axios |
